# My Arch Linux Dotfiles 

мой конфиг для Hyprland на Arch linux

- **ОС:** Arch Linux
- **Процессор:** AMD Ryzen 5 7500F
- **Видеокарта:** NVIDIA GeForce RTX 4060 Ti
- **ОЗУ:** 32GB
- **WM:** Hyprland
- **Терминал:** kitty
- **Fetch:** fastfetch

## Установка
скопируйте содержимое папки `config/*` в свою директорию `~/.config/`.
